(async) play(state, errors) → {GuildQueue}

Plays a random song from the queries specified in queries.json. This function will attempt to play again if it is unable to play due to an exception (e.g. the song cannot be found on YouTube). This will increment errors until errors is greater than 10, after which it will stop attempting to play.

Parameters:
Name Type Description
state VoiceState A VoiceState object representing the current voice state of a Guild Member
errors number The number of errors that have been encountered while attempting to play (optional)

(async) playAfterWaitTime(state, channel)

Plays a random song after a period of time. The amount of time that elapses before playback begins is equal to the amount of time specified in environment variable WAIT_TIME. Playback does not begin if the right GuildMembers are not present in the channel after the wait time has elapsed.
